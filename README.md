# Shahjalal Library Corner

Shahjalal Library Corner is a full-stack platform with:
- Public website: `frontend/slc-website`
- Admin dashboard: `admin-panel`
- Backend API: `backend`

## Local Run

- Public site: http://localhost:3010
- Backend API: http://localhost:5010
- Admin panel: http://localhost:8010

Run all with:

```powershell
.\starter.ps1
```
